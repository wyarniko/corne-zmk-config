# Eyelash Corne ZMK config

**English** | [Русский](README_RU.md)

Personal ZMK configuration for the wireless **Eyelash Corne** split keyboard.
The layout was migrated from a Jorne and adapted for macOS, the built-in
five-way switch, and CS2.

> [!IMPORTANT]
> This is not a standard foostan Corne. This keyboard requires the
> `eyelash_corne` firmware; standard `corne` firmware is not compatible.

## Layers

| Layer | Activation | Purpose |
| --- | --- | --- |
| `MAIN` | Default layer | Typing and five-way switch control |
| `SYMBOLS` | Hold the right `Enter` | Symbols, numbers, and function keys |
| `NUMBERS` | Hold the right `Del` | Numpad, navigation, and system keys |
| `MOUSE` | Hold the left `Esc` | Mouse, scrolling, and RGB controls |
| `ADJUST` | `Q + P + five-way switch press` | Bluetooth, USB, reset, bootloader, and ZMK Studio |
| `GAME` | `Y + H + N` | Left-half gaming layout |
| `GAME UTIL` | Hold the middle left thumb key on `GAME` | Weapon selection `1–5` |

A layer-tap key sends its normal keycode when tapped and activates a layer when
held. For example:

```dts
&lt SYMBOLS ENTER
```

A tap sends `Enter`; holding the key activates `SYMBOLS`.

## Combos

A combo triggers when all listed keys are pressed together.

| Keys | Result |
| --- | --- |
| `Q + W` | `Esc` |
| `Y + U` | `(` |
| `U + I` | `)` |
| `H + J` | `{` |
| `J + K` | `}` |
| `N + M` | `\|` |
| `M + ,` | `[` |
| `, + .` | `]` |
| `R + T` | `_` |
| `Y + H + N` | Toggle `GAME` |
| `Q + P + five-way switch press` | Toggle `ADJUST` |
| Outermost thumb keys | Toggle RGB |
| `Q + S + Z` held for 2 seconds | Deep sleep |

With the standard Russian macOS input layout, the `[` HID key produces `х`,
and the `]` HID key produces `ъ`.

## Five-Way Switch and Mouse

On the main layer, the five-way switch on the right half acts as a mouse:

- directions move the pointer;
- pressing the switch sends a left click.

Transparent positions on other layers inherit this behavior, although
layer-specific service bindings can override it.

On the `MOUSE` layer:

- `H/J/K/L` move the pointer Vim-style: left, down, up, right;
- the upper-right row controls scrolling;
- `Q/W` change RGB hue;
- `E/R` decrease and increase RGB brightness;
- `T` selects the next RGB effect.

RGB settings are persisted in flash storage.

## Game Layer

`GAME` is designed for CS2 using only the left keyboard half:

| Physical key | Sent key |
| --- | --- |
| `E/S/D/F` | `W/A/S/D` |
| `A` | `Left Shift` (walk) |
| `Z` | `Left Ctrl` (duck) |
| Original `Ctrl` position | `Z` |
| Three left thumb keys | `Space`, hold `GAME UTIL`, `Esc` |

On `GAME UTIL`, physical `Q/W/E/R/T` send `1/2/3/4/5`.

## Service Layer

`ADJUST` intentionally uses a difficult combo to prevent accidental resets:

- `BT_SEL 0–4` selects a Bluetooth profile;
- `BT_CLR` clears the current Bluetooth connection;
- `OUT_TOG` switches between USB and Bluetooth;
- `studio_unlock` allows changes through ZMK Studio;
- `sys_reset` restarts the controller;
- `bootloader` enters firmware flashing mode.

Press `Q + P + five-way switch` again to leave `ADJUST`.

## Building

Firmware is built automatically by
[GitHub Actions](https://github.com/wyarniko/corne-zmk-config/actions/workflows/build.yml)
after every push. The `firmware` artifact contains:

- `eyelash_corne_studio_left.uf2` — left central half;
- `eyelash_corne_right nice_view-nice_nano_v2-zmk.uf2` — right half;
- `settings_reset-nice_nano_v2-zmk.uf2` — clears saved settings.

## Flashing

1. Connect one keyboard half over USB.
2. Quickly press its physical Reset button twice.
3. Wait for the `NICENANO` drive to appear.
4. Copy the matching `.uf2` file to the root of the drive.
5. Repeat for the other half.
6. Power on both halves and press Reset once on both at the same time.

Use `settings_reset` only when troubleshooting Bluetooth or split connection
issues. Flash the regular left/right UF2 again afterwards.
